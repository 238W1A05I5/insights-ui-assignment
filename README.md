# Insights UI Assignment — Jetpack Compose

## Overview
This project implements the provided Figma design using Jetpack Compose, focusing on clean UI architecture and accurate visual representation.

## Features
- Stability Summary with smooth graph (Canvas)
- Cycle Trends visualization
- Body & Metabolic Trends (line chart)
- Symptom donut representation
- Modular and reusable composables

## Tech Stack
- Kotlin
- Jetpack Compose
- Material 3

## How to Run
1. Open project in Android Studio (Hedgehog+)
2. Sync Gradle
3. Run on emulator (API 34)
